# ERP Frontend Dashboard

A React + TypeScript ERP dashboard prototype built with Vite and Tailwind CSS. It includes a dashboard view, sidebar-based navigation, and spec-driven module screens.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Recharts

## Quick Start
```bash
npm install
npm run dev
```

App runs at `http://localhost:5173` by default.

## Scripts
- `npm run dev` - Start local dev server
- `npm run build` - Type-check and build for production
- `npm run preview` - Preview production build locally

## Project Structure
- `src/App.tsx` - App layout and route setup
- `src/components/layout` - Sidebar and top navigation
- `src/components/dashboard` - Dashboard page UI
- `src/components/spec` - Spec-driven screen renderer
- `src/data/specConfig.ts` - Module/screen configuration
